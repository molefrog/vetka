Olia Marchetti

Berlin — Creative Technologist at Feld Studio

I make things that sit at the edge between design and code. Mostly useful, occasionally beautiful.

Now

At Feld I'm deep in the interaction layer of a tool that lets city planners model neighborhoods — the kind of software where a misread color legend could mean a park ends up as a parking lot. It's teaching me to be very careful about information hierarchy, and I'm enjoying it more than I expected.

On weekends I shoot film (Minolta X-700, usually one roll a month), putter around with the Risograph, and think about the next issue of Werkzeug. It's overdue. I have notes.

Work

  • Pairs A font pairing tool I built in a weekend. Still running, somehow. Around 2k monthly users, which continues to surprise me.
  • Moss An ambient soundscape generator that reads live weather data from your location and turns it into something you can leave running in the background. Built with Web Audio API.
  • Werkzeug A newsletter about tools — the ones we use to make things, and how they shape what we end up making. Six issues a year, 800 readers. The name is German for "tool."
